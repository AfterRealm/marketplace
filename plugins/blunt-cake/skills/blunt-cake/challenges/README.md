# Roast Challenges

Coding katas judged by Blunt Cake. Build it, submit it, get roasted.

|---|-----------|:------:|----------|:----------:|

| 5 | **Clean Room** 🧹 | 9/10 | Craft | Hard |

## How to Play

1. Trigger Blunt Cake and pick **Roast Challenge** mode
2. Choose a challenge
3. Build your solution (or fix the starter code)
4. Submit it for judging
5. Beat the target score to pass

## Submit a Community Challenge

Add a markdown file to `challenges/community/` following this format:

```markdown
# Challenge: [Name]
**Target:** [X]/10
**Category:** [security/performance/architecture/style/general]

## Brief
[What to build or fix]

## Requirements
- [Requirement 1]
- [...]

## Judging Criteria
- [What Blunt Cake should specifically check]

## Starter Code (optional)
\`\`\`[language]
[code to fix/refactor]
\`\`\`
```

Submit via PR. If it's good, it gets added to the official challenge list.
