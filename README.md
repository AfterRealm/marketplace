# AfterRealm Marketplace

Claude Code plugins by [AfterRealm](https://github.com/AfterRealm).

## Install the Marketplace

```bash
claude marketplace add AfterRealm/marketplace
```

## Available Plugins

### Blunt Cake

Brutal, funny code reviewer with 7 modes and 6 personalities. Every finding is real, every roast comes with a fix.

```bash
claude plugin add afterrealm/blunt-cake
```

**Modes:** Standard Roast, Panel Roast (multi-agent), Skill Roast, Eval Mode, Diff Roast, Batter Battle, Roast-a-thon

**Personalities:** Chef, Disappointed Grandma, Passive-Aggressive PR Reviewer, Simon Cowell, Snoop Dogg, Pirate

[Full README](https://github.com/AfterRealm/blunt-cake) | v2.1.0

---

### Father Time

Session-aware time management for Claude Code. Tracks session duration, context usage, and helps you work with your schedule instead of against it.

```bash
claude plugin add afterrealm/father-time
```

**Skills:** Time Menu, Session Timer, Session Health, Peak Hours, Daily Brief, Focus Mode, Pace Check, Context Budget, Activity Patterns

[Full README](https://github.com/AfterRealm/father-time) | v1.8.1
